# 商品管理システム

## 概要

このシステムでは商品の管理が行えます。

商品の新規登録から編集、削除を行うことが出来ます。

## 主な機能

- ログイン・ログアウト機能
- 商品一覧画面
- 商品新規登録、編集、削除機能

## 開発環境
```
PHP 8.2.28 
MySQL 8.0.40
Laravel 10.13.5
```
